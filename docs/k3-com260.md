K3 CoM260
===========

The K3 CoM260 is a 64-bit, RVA23-compliant RISC-V platform based on the
SpacemiT K3 SoC.

How to Build
============

Clone and enable these repositories and enable the below layers:

* bitbake
* openembedded-core
  * meta
* meta-riscv

See [the Yocto Project](https://docs.yoctoproject.org/brief-yoctoprojectqs/index.html) manual for details.

Set these variables in a configuration file:

* `MACHINE = "k3-com260"`
* `EXTRA_IMAGE_FEATURES = "allow-empty-password empty-root-password allow-root-login post-install-logging"`

Build your image:

```
$ bitbake core-image-minimal
```

Flashing the Image
==================

Flash your image to a microSD card with `bmaptool`:

```
$ sudo bmaptool copy build/tmp/deploy/images/k3-com260/core-image-minimal-k3-com260.rootfs.wic.gz /dev/sdx
```

Boot the Board
==============

Insert the microSD card, then connect a USB to serial dongle to your board and
to your PC, and start your favorite terminal emulator:

```
$ sudo apt install tio
$ tio /dev/ttyUSB0
```

Power the board and you will see it boot to a Linux command line shell.
